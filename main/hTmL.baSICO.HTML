Estructura básica de HTML

HTML (HyperText Markup Language) es el lenguaje estándar para crear páginas web. Cada documento HTML sigue una estructura básica que consta de varias etiquetas que proporcionan instrucciones al navegador sobre cómo mostrar el contenido.

La estructura básica de un documento HTML incluye las siguientes etiquetas:

  • <!DOCTYPE html>: Declara la versión de HTML utilizada en el documento.
  • <html lang="es">: Envuelve todo el contenido HTML del documento y especifica que el idioma utilizado es español.
  • <head>: Contiene metadatos del documento, como el título de la página, enlaces a hojas de estilo y scripts, entre otros.
  • <meta charset="UTF-8">: Especifica el conjunto de caracteres utilizado en el documento, generalmente UTF-8, que es compatible con la mayoría de los idiomas y caracteres especiales.
  • <title>: Define el título de la página que se muestra en la pestaña del navegador.
  • <body>: Contiene el contenido visible de la página, como texto, imágenes, enlaces, etc.

A continuación, se muestra un ejemplo de una estructura básica de HTML:


<!DOCTYPE html>
<html lang="es"> 
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Título de la página</title>
</head>
<body>
    <h1>Mi primera página web</h1>
    <p>¡Hola, mundo!</p>
</body>
</html>

Etiquetas HTML

Las etiquetas HTML son como instrucciones que le dices al navegador sobre cómo mostrar una página web. Cada etiqueta tiene un propósito específico y ayuda a organizar y estructurar el contenido de la página.

Algunas etiquetas HTML comunes son:

  • <p>: Se utiliza para crear párrafos de texto.
  • <h1> a <h6>: Son para títulos y subtítulos de diferentes niveles de importancia.
  • <a>: Se emplea para crear enlaces a otras páginas web o recursos.
  • <img>: Sirve para insertar imágenes en la página.
  • <ul>: Se usa para hacer listas desordenadas, como una lista de compras.
  • <ol>: Es para hacer listas ordenadas, como una lista de pasos.
  • <li>: Representa cada elemento de una lista.
  • <div>: Ayuda a agrupar y estructurar contenido de manera flexible.
  • <span>: Se utiliza para aplicar estilos o interactividad a partes específicas del texto.

Por ejemplo, para crear un párrafo de texto en HTML, puedes usar la etiqueta <p> de la siguiente manera:

<p>Este es un párrafo de ejemplo.</p>

Atributos HTML

Los atributos proporcionan información adicional sobre una etiqueta HTML. Se utilizan para personalizar el comportamiento o la apariencia de la etiqueta.

Por ejemplo:

  • El atributo href se utiliza en la etiqueta <a> para especificar la URL de destino de un enlace.

    <a href="https://www.ejemplo.com">Enlace de ejemplo</a>

    En este ejemplo, el atributo href contiene la dirección web a la que se dirige el enlace.

  • El atributo src se utiliza en la etiqueta <img> para especificar la ruta de la imagen que se mostrará.

    <img src="ruta_de_la_imagen.jpg" alt="Descripción de la imagen">

    En este caso, el atributo src indica la ubicación de la imagen que se mostrará en la página.

  • El atributo alt en la etiqueta <img> proporciona un texto alternativo que se muestra si la imagen no se puede cargar o para usuarios con discapacidad visual.

    <img src="ruta_de_la_imagen.jpg" alt="Descripción de la imagen">

    En este ejemplo, el atributo alt contiene una breve descripción de la imagen.

  • El atributo title se puede usar en varias etiquetas, como <a> o <img>, para proporcionar información adicional que se muestra al pasar el ratón sobre el elemento. 

    <a href="https://www.ejemplo.com" title="Enlace de ejemplo">Enlace de ejemplo</a>

    En este caso, el atributo title contiene un texto que se muestra como una descripción emergente cuando el cursor se sitúa sobre el enlace.

Estructura de la página web

La estructura de una página web es crucial para su legibilidad, accesibilidad y SEO (optimización para motores de búsqueda). HTML5 ofrece una serie de etiquetas semánticas que ayudan a organizar el contenido de manera clara y significativa.

Algunas etiquetas semánticas comunes incluyen:

  • <header>: Define la cabecera de la página. Se utiliza para colocar elementos como el logo, el título del sitio y el menú de navegación.
  • <nav>: Define la sección de navegación. Es útil para agrupar enlaces que guían a otras partes del sitio.
  • <main>: Define el contenido principal de la página. Aquí se encuentra la información más relevante y sustancial para los visitantes.
  • <section>: Define una sección genérica en el documento. Se utiliza para agrupar contenido relacionado temáticamente.
  • <article>: Define un contenido independiente y autocontenido, como un artículo, un comentario o una publicación en un blog.
  • <aside>: Define contenido que está tangencialmente relacionado con el contenido que lo rodea, como barras laterales o bloques de anuncios.
  • <footer>: Define el pie de página de la página. Contiene información de contacto, enlaces adicionales y derechos de autor.

Estas etiquetas ayudan a los motores de búsqueda y a los lectores de pantalla a comprender la estructura y el contenido de la página.

A continuación, se muestra un ejemplo de cómo utilizar algunas de estas etiquetas:

<header>
                <h1>Título del sitio</h1>
                <nav>
                    <ul>
                        <li><a href="#">Inicio</a></li>
                        <li><a href="#">Acerca de</a></li>
                        <li><a href="#">Contacto</a></li>
                    </ul>
                </nav>
            </header>
            <main>
                <section>
                    <h2>Título de la sección</h2>
                    <p>Contenido de la sección.</p>
                </section>
            </main>
            <footer>
                <p>Pie de página</p>
            </footer>

Enlaces y anclajes

Enlaces

Los enlaces son fundamentales para la navegación web, permitiendo a los usuarios acceder a diferentes páginas y recursos. En HTML, los enlaces se crean utilizando la etiqueta <a>, con el atributo href especificando la URL de destino.

Para enlazar a una sección específica de una página, se utilizan los anclajes. Se pueden crear anclajes utilizando la etiqueta <a> con el atributo href apuntando al ID de la sección.

A continuación, se muestra un ejemplo de cómo crear un enlace y un anclaje:

<a href="https://www.ejemplo.com">Enlace externo</a>
            <a href="#seccion-1">Ir a la sección 1</a>
            
            ...
            
            <h2 id="seccion-1">Sección 1</h2>

Imágenes

Las imágenes son componentes importantes de una página web, ya que pueden mejorar la estética y la comprensión del contenido. En HTML, las imágenes se insertan utilizando la etiqueta <img>.

La etiqueta <img> requiere el atributo src, que especifica la ruta de la imagen, y opcionalmente los atributos alt, width y height.

A continuación, se muestra un ejemplo de cómo insertar una imagen:

<img src="imagen.jpg" alt="Descripción de la imagen" width="100" height="100">

Listas

Las listas son herramientas útiles para organizar y presentar información de forma estructurada en una página web. En HTML, existen dos tipos principales de listas:

  • Listas ordenadas: Utiliza la etiqueta <ol> para crear una lista numerada.
  • Listas no ordenadas: Utiliza la etiqueta <ul> para crear una lista con viñetas.

Para cada elemento de la lista, se utiliza la etiqueta <li>.

A continuación, se muestra un ejemplo de cómo crear una lista ordenada y una lista no ordenada:

<ol>
                <li>Elemento 1</li>
                <li>Elemento 2</li>
            </ol>
            
            <ul>
                <li>Elemento A</li>
                <li>Elemento B</li>
            </ul>

Aquí hay más ejemplos:

Listas Ordenadas:

  1. Manzanas
  2. Plátanos
  3. Naranjas

Listas No Ordenadas:

  • Perros
  • Gatos
  • Pájaros

Formularios básicos

Los formularios son elementos interactivos que permiten a los usuarios enviar datos al servidor para procesamiento. En HTML, los formularios se crean utilizando la etiqueta <form>.

Dentro de un formulario, se pueden incluir diferentes tipos de elementos para que los usuarios ingresen información, como campos de texto, áreas de texto, botones de opción y botones de envío.

Ejemplo de un formulario básico:

<form action="/procesar-datos" method="post">
    <label for="nombre">Nombre:</label>
    <input type="text" id="nombre" name="nombre">
    <input type="submit" value="Enviar">
</form>

En este ejemplo:

  • La etiqueta <form> delimita el inicio y el final del formulario.
  • El atributo action especifica la URL del script de servidor que manejará los datos enviados por el formulario.
  • El atributo method indica el método HTTP utilizado para enviar los datos, que puede ser post o get.
  • El elemento <input> con type="text" crea un campo de entrada de texto para que el usuario ingrese su nombre.
  • El elemento <input> con type="submit" crea un botón de envío que enviará los datos del formulario al servidor.

Otros ejemplos de elementos de formulario:

  • <input type="password">: Crea un campo de contraseña.
  • <textarea>: Crea un área de texto de varias líneas para que el usuario escriba un mensaje largo.
  • <input type="checkbox">: Crea una casilla de verificación que el usuario puede marcar o desmarcar.
  • <input type="radio">: Crea un botón de opción que el usuario puede seleccionar.
  • <select>: Crea un menú desplegable con opciones predefinidas.

Estructura de tablas

Las tablas son útiles para organizar datos en filas y columnas. En HTML, las tablas se crean utilizando las etiquetas <table>, <tr>, <th> y <td>.

La etiqueta <table> define la tabla en sí, <tr> define una fila, <th> define una celda de encabezado y <td> define una celda de datos.

A continuación, se muestra un ejemplo de cómo crear una tabla:

<table>
                <tr>
                    <th>Encabezado 1</th>
                    <th>Encabezado 2</th>
                </tr>
                <tr>
                    <td>Dato 1</td>
                    <td>Dato 2</td>
                </tr>
            </table>

Comentarios HTML

Los comentarios son útiles para agregar notas o explicaciones en el código HTML sin que se muestren en la página web. En HTML, los comentarios se crean utilizando la sintaxis <!-- comentario -->.

Los comentarios son útiles para hacer anotaciones para ti mismo o para otros desarrolladores que trabajen en el mismo proyecto.

A continuación, se muestra un ejemplo de cómo agregar un comentario en HTML:

<!-- Este es un comentario en HTML -->
